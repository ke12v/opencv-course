# Colab setup: `basic_functions.py` notebook

## 1) Upload / mount the repo
In Google Colab:
- Either **Upload** this entire folder (recommended), or
- **Git clone** the repo into the Colab environment.

This notebook expects these relative paths to exist:
- `Resources/Photos/park.jpg`
- `Section 1 output/`

## 2) (Optional) Install OpenCV
Uncomment the `pip install opencv-python` line in the first cell if OpenCV is missing.

## 3) Run cells
Run from top to bottom.

## 4) Outputs
Images are saved to:
- `Section 1 output/basic_functions_*.png`

and also displayed inline in the notebook.
